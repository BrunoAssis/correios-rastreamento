h2. Usando a API Ruby

<pre>
require "rubygems"
require "correios"
encomenda = CorreiosRastreamento.encomenda("ES446391025BR")
puts encomenda.numero
encomenda.status.each do |status|
  puts status.data
  puts status.local
  puts status.situacao
  puts status.detalhes
end
</pre>
